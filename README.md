# Fiber Laser GUI Simulation

Simulated motor and power meter control GUI for fiber laser experiments.

---

## Features

- Real-time power measurement plots (S1, S2, Ratio)
- Motor control with position feedback
- Experiment automation and data saving
- GUI implemented in PyQt5 with matplotlib

---

## Requirements

- Python 3.10+
- PyQt5
- matplotlib
- pyvisa

Install dependencies:

```bash
pip install pyqt5 matplotlib pyvisa
